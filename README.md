# Quickstart React Template

A minimal React 18 starter with Express backend, testing, and CI ready.

## Getting Started

```bash
npm install
npm start
```

Runs the dev server on http://localhost:3000

## Testing

```bash
npm test
```

## Structure

```
src/          - React components
server/       - Express API backend
test/         - Jest test suite
scripts/      - Build & setup utilities
```
